Topic Structure

üì• Subscribe Topics (Receive Data)

Topic Description Payload Example

Example Client Code

Python (paho-mqtt)

import paho.mqtt.client as mqtt

client = mqtt.Client()
client.connect("localhost", 1883, 60)

# Subscribe to telemetry
client.subscribe("devices/+/telemetry")

# Publish GPIO command
client.publish("device/ESP32-IOT-ACTUATOR-4e5c/gpio/set", '{"gpio":1,"state":1}')

client.loop_forever()
